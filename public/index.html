How it runs

From a company name to a reviewed email.

Two AI agents share one Gemini model. The first does research, the second writes the email. Two defensive branches catch bad data so a thin scrape or malformed JSON never produces a garbage email.

How does this work?

When you submit the form:

  1. Data is sent to n8n via Cloudflare Tunnel
  2. n8n scrapes the company website and extracts text
  3. Research Agent analyses the content (Gemini)
  4. Email Writer creates a personalised draft (Gemini)
  5. The draft is returned as JSON to this page
  6. You review and edit inline — no redirect
  7. On confirm, a second webhook sends the email via Gmail SMTP
  8. Everything is logged to Google Sheets
